**Chiến dịch Heap-Nullable** — `T?` cho T heap (String/Vector/HashMap). ADR-0062 (`docs/decisions/0062-heap-nullable-ptr-sentinel-repr.md`) LOCKED, O+G ký (`7fd3937`). Repr (a) **ptr-sentinel**: `T?` dùng CÙNG slot với `T`; null = ô `ptr == NULL_SENTINEL` (`i64::MIN`), CẤM `ptr==0`; widening = no-op; 0 byte overhead. Scope khóa String/Vector/HashMap; **defer Struct?/Enum?** (aggregate đa-word, không có ô ptr tự nhiên — cần ADR riêng).
